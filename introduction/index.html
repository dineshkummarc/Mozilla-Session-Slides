    So what the heck's all that

  • Firefox - No explanation required
  • Thunderbird - An email client with intelligent pam filters, powerful search and customizable views
  • Seamonkey - All-in-one application, contains a web browser, mail, newsgroups client, an HTML editor and an IRC Client
  • Lightning and Sunbird - Lightning is a popular calendaring, scheduling and task management extension. Sunbird is a cross-platform application that brings Mozilla-style ease-of-use to your calendar.
  • Camino - a Web browser optimized for Mac OS X with a Cocoa user interface, and powerful Gecko layout engine.
  • Labs - a place where people come together to create, experiment and play with new Web innovations and technologies.
